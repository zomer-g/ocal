## רשימת ביקורת לפני אישור הפרסום

לפני שמודיעים שהעמודים מוכנים, יש לוודא:

- [ ] שלושת ה-URLs פעילים ומחזירים HTTP 200 בלי לוגאין.
- [ ] תאריך העדכון בראש עמודי הפרטיות והתנאים מולא בפועל (ולא נשאר `__[יש למלא...]__`).
- [ ] בעמוד `/ocal`, הקישור ל-Chrome Web Store עדיין placeholder או הוחלף ב-Live URL לאחר אישור.
- [ ] בכל עמוד יש `<html lang="he" dir="rtl">`.
- [ ] התצוגה הניידת תקינה (RTL לא נשבר).
- [ ] כתובת הדוא"ל המופיעה זהה לכתובת הנמצאת בעמוד `legal-tools` הקיים.
- [ ] קישורים פנימיים בין שלושת העמודים עובדים (`/ocal` ↔ `/ocal-privacy` ↔ `/ocal-terms`).
- [ ] כותרות `<title>` ו-meta descriptions מולאו לכל אחד מהעמודים.
